Overview

The Little Lemon Menu & Booking API offers a collection of endpoints to manage user accounts, menu items, and reservations supports user registration, token-based authentication, and various managerial functions.

Features

User Management

Registration and authentication of consumers

Inventory

CRUD operations on menu items

Performance Optimizations and Monitoring

While not required by the assignment. This API includes rate limits, key metrics monitoring and static asset compression and optimization.

Reservation Management

CRUD Operations on reservations

Getting Started & API Documentation

The API follows the OpenAPI 3.0 standards, which is a powerful tool for designing, documenting, and developing APIs. It provides a standardized format for API specifications, which enables design-first development, automated documentation, code generation, and test automation.

Explore the Full API Schema

The API is organized into several categories, each serving a specific purpose. Here are the main route categories:

  • Auth – User registration, authentication, and token management.
  • Menu – CRUD operations for menu items.

    • UML Class Diagram For or Menu Model

                        classDiagram

                      class Menu {
                        +SmallAutoField item_id : Integer
                        +CharField title : String
                        +DecimalField price : Decimal
                        +PositiveSmallIntegerField inventory : Integer
                            }
                                Menu : Meta
                            Menu : - app_label = "resturant"
                            Menu : - db_table = "inventory"
                            Menu : - ordering = ["title"]
                            Menu : - indexes = [price, price+inventory, title, title+inventory]
  • Bookings – CRUD operations for reservations.

    • UML Class Diagram For or Booking Model

                classDiagram
    class Booking {
        +SmallAutoField booking_id
        +CharField name
        +PositiveSmallIntegerField no_of_guests
        +DateTimeField date
        +__str__(): str
    }

    Booking : Meta
    Booking : - app_label = "resturant"
    Booking : - db_table = "bookings"
    Booking : - ordering = ["-date"]
    Booking : - indexes = [name, date]
  • Users – User management and profile operations.
